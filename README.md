# psy120L-final-report
It’s Not You, It’s Your Parents: Parental Warmth, Academic Feedback, and Student Motivation
This repository contains the published HTML version of my PSY 10B final reproducible report.
This project examined whether autonomy-supportive academic feedback increased student achievement motivation and self-efficacy, and whether perceived parental warmth moderated these effects.
The study used a 2 × 2 experimental design comparing Control and Autonomy-Supportive feedback conditions alongside High vs. Low parental warmth groups.
Analyses were conducted in R using Quarto to create a fully reproducible report.
The dataset is not included in this repository.
